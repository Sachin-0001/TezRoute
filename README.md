### 1. **Navigate**
```bash
cd backend
```

### 2. **Install Packages**
```bash
pip install -r requirements.txt
```

### 2. **Run the application**
```bash
uvicorn app:app --reload --port 8000
```

The backend will be running at `http://localhost:8000`
